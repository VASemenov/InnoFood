## Roles

| Role              | Description                                                  |
| ----------------- | ------------------------------------------------------------ |
| **Administrator** | Person that is responsible for creating and administrating cafes (assigning managers, changing contact data and info). |

| **Customer**          | Person which orders food.                                    |
